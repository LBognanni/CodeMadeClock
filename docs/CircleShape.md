# CircleShape

Represents a circle with position, radius and color



## Properties
### Position

Represents the center of the circle



See [Vertex](Vertex.md)
### Radius

Circle radius



### Color

Circle color. Can be a solid color or a gradient.



See [Colors](Colors.md)

---

## Example

```json
{
    "$type": "CircleShape",
    "Position": {
        "X": 49,
        "Y": 49
    },
    "Radius": 43,
    "Color": "30-#59fb-#358b"
}

```
